debugMode([enable]);

Toggle a helpful mode with 3D Graphics

Arguments

[enable]Whether or not to enable the debug mode, if not provided, it defaults to true

  createWindow(800, 800, GL3D);

  debugMode();
end

function draw()
  background(51);
  
  box(100);
end
function setup()
  createWindow(800, 800, GL3D);

function draw()
  background(51);
  
  -- When the d key is down, show debug graphics
  if (keyIsDown('d')) then
    debugMode(true);
  else
    debugMode(false);
  end

  box(100);
end
