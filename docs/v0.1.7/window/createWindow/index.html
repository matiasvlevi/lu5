createWindow

v0.1.7
createWindow(w, h, [mode]);

Create a GLFW window.

Arguments

wWindow width
hWindow height
[mode]Rendering mode, either GL2D or GL3D, by default GL2D is set.

function setup()
  createWindow(100, 100);
end

function draw()
  background('magenta');
  -- draw things
end
