Vector.mult

v0.1.7
Vector.mult(a, b);

No description

Arguments

aThe first vector
bThe second vector

Returns

VectorThe multiplied vector

local a = createVector(3, 5);
local b = createVector(2, 3);

-- Non-mutable
print(Vector.mult(a, b)); -- { 6.0, 15.0 }
print(a * b);             -- { 6.0, 15.0 }
print(a);                 -- { 3.0, 5.0 }

-- Mutable
print(a:mult(b)); -- { 3.0, 15.0 }
print(a);         -- { 3.0, 15.0 }
