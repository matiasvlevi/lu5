Vector.idiv

v0.1.7
Vector.idiv(a, b);

No description

Arguments

aThe first vector
bThe second vector

Returns

VectorThe divided vector

local a = createVector(7, 8);
local b = createVector(2, 3);

-- Non-mutable
print(Vector.idiv(a, b)) -- { 3.0, 2.0 }
print(a // b);           -- { 3.0, 2.0 }
print(a);                -- { 7.0, 8.0 }

-- Mutable
print(a:idiv(b)); -- { 3.0, 2.0 }
print(a)          -- { 3.0, 2.0 }
