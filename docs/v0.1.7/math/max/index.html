max

v0.1.7
max(a, b);

Return the largest value.
Can also take any number of arguments, or a table.

Arguments

aThe first value
bThe second value

Returns

numberThe largest value

max(2, 5); -- returns 5

max(2, 5, 7); -- returns 7

max({ 2, 5, 7 }); -- returns 7
See max in lu5_math.h
