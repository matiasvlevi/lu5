loaders

v0.0.3
loadFont(path);

Load a font face

Arguments

pathThe font path on the local system

Returns

fontThe image reference

 function setup()
   createWindow(600, 600);
   myfont = loadFont('/path/to/myfont.ttf');
 end

 function draw()
   background(51);

   textFont(myfont);
   text('Hello from lu5!', 30, 50);
 end

loadImage(path);

Load a png or jpeg image

Arguments

pathThe local image path

Returns

imageThe image reference

 function setup()
   createWindow(600, 600);
   img = loadImage('/path/to/mypic.png');
 end

 function draw()
   background(51);

   image(img, 0, 0);
 end
