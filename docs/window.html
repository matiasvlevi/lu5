window

v0.0.3
createWindow(w, h);

Create a GLFW window.

Arguments

wWindow width
hWindow height

It is also possible to create a window in the global scope without defining a setup function.

 function setup()
   -- Create the window here
   createWindow(600, 600);
 end

 function draw()
   -- draw things
 end
